# Website Resmi HMJ Teknik Informatika UNMUS

## Deskripsi Proyek

Repository ini merupakan repository resmi untuk pengembangan **Website Himpunan Mahasiswa Jurusan (HMJ) Teknik Informatika Universitas Musamus**.  
Website ini bertujuan sebagai media informasi, dokumentasi, dan publikasi kegiatan organisasi HMJ Teknik Informatika.

## Tujuan Proyek

Tujuan dikembangkannya website ini antara lain:
1. Menyediakan informasi resmi terkait HMJ Teknik Informatika.
2. Menjadi media publikasi kegiatan dan program kerja organisasi.
3. Menjadi sarana dokumentasi struktur organisasi serta visi dan misi.
4. Meningkatkan transparansi dan akses informasi bagi mahasiswa.

## Cara Menjalankan Proyek

### Pengembangan Secara Lokal

Proyek ini dapat dijalankan secara lokal menggunakan IDE pilihan masing-masing.

**Persyaratan Sistem:**
- Node.js
- npm (Node Package Manager)

**Langkah-langkah instalasi:**

```sh
# Clone repository
git clone <URL_GIT_REPOSITORY>

# Masuk ke direktori proyek
cd <NAMA_FOLDER_PROYEK>

# Install dependensi
npm install

# Menjalankan server pengembangan
npm run dev
